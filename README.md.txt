# Phone_Pey Analyst Dashboard

## Overview
 Power BI dashboard provides interactive insights into Phone_Pey transaction data. It analyzes transaction volume, transaction value, user activity, payment status, service performance, and customer behavior to support business decision-making.

## Features
- Total Transactions
- Total Transaction Value
- Unique Users
- Success Rate
- Transaction Trend Analysis
- Service-wise Analysis
- Age Group Contribution
- Top 5 Users
- Weekday vs Weekend Usage

## Tools Used
- Power BI
- DAX
- Power Query
- Excel
